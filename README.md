# 🛂 SCRUM GUARD — スクラムの門番

スクラムの知識を楽しく学べるブラウザゲーム。

あなたはスクラムゲートの検問官。正しいスクラムを実践するチームを **通過許可** し、アンチパターンを踏むチームを **連行** せよ！

## 🎮 遊び方

1. チームの申請書とセリフを読む
2. スクラムガイドの確認項目を参考に判断
3. **✓ 通過許可** か **✗ 連行** を選ぶ
4. 3回ミスでゲームオーバー

## 🧠 学べること

全21シナリオで以下のスクラムの知識をカバー：

- デイリースクラムの正しい運用
- スプリント中のスコープ保護
- スクラムマスターの役割（サーバントリーダー）
- レトロスペクティブと改善アクション
- Definition of Done（完成の定義）
- プロダクトバックログの管理
- ベロシティと予測可能性
- クロスファンクショナルチーム
- 「スクラム開発」と「スクラムを用いた開発」の違い
- ...など

ジェフ・サザーランドやケン・シュエイバーの本物と偽物を見分ける問題も！

## 🚀 起動方法

静的ファイルのみで動作します。任意のHTTPサーバーで配信してください。

```bash
# 例
npx serve .
# または
python3 -m http.server 8000
```

`index.html` をブラウザで開くだけでもOKです。

## 📁 ファイル構成

```
index.html      - メインHTML
style.css       - スタイル（PC/モバイル対応）
game.js         - ゲームロジック・アニメーション制御
scenarios.js    - 全21シナリオ・ルールデータ
sounds/         - サウンド素材
```

## 📱 対応環境

- PC / スマートフォン両対応（レスポンシブ）
- モダンブラウザ（Chrome, Safari, Firefox, Edge）
